# MVP動作確認メモ

## 実行日
2025-08-16 17:51 JST

## テスト内容
- トリガー「セキララ」をグループに送信
- 2人がそれぞれ最初の回答を投稿
- AIが要約を1回だけ返す

## 期待する結果
1. 最初の回答のみ保存される（2回目以降は無視）
2. 要約は1通だけ返る（追加回答では再要約しない）
3. 要約には両者の回答が含まれる
4. 要約の末尾に次の一歩（例：「自宅でチャイを作ってみる」）が提案される

## 実際の結果
- 期待通りの動作を確認
- 要約に両者の内容が含まれ、提案もあり
- DB `answers` に2レコード、`ai_summaries` に1レコードのみ保存されているのを確認
- SQL結果は `answers_sample.sql.txt` を参照
